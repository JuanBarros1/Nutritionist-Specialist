# Nutrition Specialist - Professional Wellness Landing Page

A premium, modern, and highly interactive landing page designed for a professional Nutritionist. This project showcases a sleek user interface with a focus on user experience, performance, and professional aesthetics.

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen.svg)](https://nutritionspecialist.netlify.app)

##  Live Site
You can visit the live application here: **[nutritionspecialist.netlify.app](https://nutritionspecialist.netlify.app)**

##  Features

- **Premium Design**: A clean, modern aesthetic tailored for health and wellness professionals.
- **Micro-Animations**: Smooth interactions powered by **Framer Motion** and **GSAP**.
- **Responsive Layout**: Optimized for all devices, from mobile phones to high-resolution desktops.
- **Dynamic Sections**:
  - **Hero**: Atmospheric intro with clear Call-to-Action.
  - **About**: Professional profile and mission.
  - **Services**: Detailed breakdown of nutritional programs.
  - **Journey**: Interactive career and methodology timeline.
  - **Testimonials**: Social proof and client success stories.
  - **Contact**: Integrated contact options for personalized consultations.
- **Fast Performance**: Built with **Vite** for near-instant load times.

##  Tech Stack

- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) (shadcn/ui)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

##  Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or bun

### Installation

1. **Clone the repository**:
   ```sh
   git clone https://github.com/JuanBarros1/Nutritionist-Specialist.git
   cd Nutritionist-Specialist
   ```

2. **Install dependencies**:
   ```sh
   npm install
   # or
   bun install
   ```

3. **Start the development server**:
   ```sh
   npm run dev
   ```

4. **Build for production**:
   ```sh
   npm run build
   ```
